# etylizer - ICFP '26 experience report

This is the project that can type check a frozen source version of etylizer
erlang_types.

Two caveats:
* widening to number() flag is not yet implemented
* binary types not yet merged, need to ignore two functions because of that

Other than that,

typecheck_frozen_erlang_types.sh

is used to generate the results for the paper.
